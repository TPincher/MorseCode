Morse code translator

Instructions

You can use this site to convert a message between English and Morse code.

To convert from English to Morse code, just start typing! Your converted message will appear on the right-hand panel.

You can use the Morse code library below to create your own messages, too. Just start typing in Morse code in the box above and your message will be automatically converted for you. Don't forget that letters in Morse code are separated with a ' ' and words are separated by a single '/'

If you're entering a Morse code message, make sure to only use dots and dashes!
